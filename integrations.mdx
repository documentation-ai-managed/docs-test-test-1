## Setting Up Webhooks for Notifications

Webhooks deliver real-time events from Test test to your endpoint. Configure them to notify teams on page updates, new versions, or user actions.

<Callout kind="info">
  Ensure your webhook endpoint accepts `POST` requests with JSON payloads and responds with `200 OK` within 5 seconds.
</Callout>

Follow these steps to create a webhook:

<Steps>
  <Step title="Create Webhook" icon="plus">
    Log in to your Test test dashboard at `https://dashboard.example.com`.
    
    Navigate to **Settings > Integrations > Webhooks**.
    
    Click **New Webhook** and enter your endpoint URL, such as `https://your-webhook-url.com/testtest`.
  </Step>
  
  <Step title="Select Events" icon="settings">
    Choose events like `page.updated` or `space.exported`.
    
    Test the webhook with the **Send Test** button.
  </Step>
  
  <Step title="Verify Payload" icon="check-circle">
    Handle incoming payloads securely. Example structure:
    
````json
{
  "event": "page.updated",
  "space_id": "spc_123abc",
  "page_id": "pg_456def",
  "timestamp": 1699123456
}
````
  </Step>
</Steps>

Handle webhooks in your server:

<CodeGroup tabs="Node.js,Python">
```javascript
const express = require('express');
const app = express();
app.use(express.json());

app.post('/testtest', (req, res) => {
  const { event, space_id, page_id } = req.body;
  console.log(`Event: ${event} for space ${space_id}`);
  // Process notification
  res.status(200).send('OK');
});
```
```python
from flask import Flask, request, jsonify
app = Flask(__name__)

@app.route('/testtest', methods=['POST'])
def webhook():
    data = request.json
    event, space_id, page_id = data['event'], data['space_id'], data['page_id']
    print(f"Event: {event} for space {space_id}")
    # Process notification
    return jsonify({'status': 'OK'}), 200
```
</CodeGroup>

## API Access for Custom Connections

Build custom integrations using Test test's REST API at `https://api.example.com/v1`. Authenticate with `YOUR_API_KEY`.

<Request tabs="cURL,JavaScript" show-lines="true">
```bash
curl -X GET https://api.example.com/v1/spaces \
  -H "Authorization: Bearer YOUR_API_KEY"
```
```javascript
const response = await fetch('https://api.example.com/v1/spaces', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});
const spaces = await response.json();
```
</Request>

<ParamField path="space_id" param-type="string" required="true">
  Unique space identifier, e.g., `spc_123abc`.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>
